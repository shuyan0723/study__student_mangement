# 学生成绩管理系统 - 功能详细说明

## 📚 功能清单

### 一、登录与认证模块

#### 功能描述
- **多角色登录**: 支持学生、教师、管理员三种角色登录
- **快速演示**: 提供快速登录按钮，方便测试不同角色
- **认证管理**: 基于 JWT Token 的身份认证
- **会话管理**: 自动登出和 Token 过期处理

#### 实现特性
- ✅ 用户名密码验证
- ✅ 记住登录状态（LocalStorage）
- ✅ 安全的密码处理
- ✅ 演示账号快速登录

### 二、学生功能模块

#### 2.1 成绩查询 📊
**功能描述**: 学生可以查看自己的所有课程成绩

**核心功能**:
- ✅ 查看所有课程成绩列表
- ✅ 实时计算平均成绩
- ✅ 统计优秀课程数量
- ✅ 统计不及格课程数量
- ✅ 查看教师评语
- ✅ 按成绩排序
- ✅ 导出成绩单 (预留接口)

**技术实现**:
```typescript
- 使用 Ant Design Table 展示数据
- 实时计算统计数据
- 支持多条件排序
- 成绩等级颜色标签区分
```

#### 2.2 选课管理 📖
**功能描述**: 学生可以浏览、选择、退课

**核心功能**:
- ✅ 查看可选课程列表
- ✅ 课程详细信息展示（学分、学时、学期、授课教师）
- ✅ 选课功能
- ✅ 查看已选课程
- ✅ 退课功能
- ✅ 课程状态管理（学习中、已完成、不及格）
- ✅ 防止重复选课

**课程状态**:
- 学习中: 正在进行的课程
- 已完成: 已通过的课程
- 不及格: 未通过的课程

#### 2.3 在线交流 💬
**功能描述**: 学生与教师的实时消息交互

**核心功能**:
- ✅ 查看教师列表
- ✅ 选择指定教师进行交流
- ✅ 发送消息
- ✅ 接收消息
- ✅ 消息时间戳
- ✅ 消息已读状态跟踪
- ✅ 对话历史记录

**消息特性**:
- 实时通知
- 消息分类（发送/接收）
- 时间格式化显示
- 未读消息标记

#### 2.4 首页仪表板 📈
**功能描述**: 学生个性化首页

**展示内容**:
- 欢迎信息
- 平均成绩统计
- 已修课程数
- 已评分课程数
- 优秀课程数
- 最近5条成绩记录
- 最近3门选修课程

## 🎯 核心特性

### 1. 多角色权限管理
- 基于 JWT Token 的身份认证
- 路由级别的权限保护 (ProtectedRoute)
- 不同角色显示不同菜单和功能

### 2. 完整的 CRUD 操作
- Create: 添加学生、教师、课程、成绩
- Read: 查看详细信息和列表
- Update: 编辑学生、教师、课程、成绩
- Delete: 删除学生、教师、课程、成绩

### 3. 数据状态管理
- 使用 Zustand 轻量级状态管理
- 清晰的数据流向
- 易于维护和扩展

### 4. 用户体验
- 现代化的渐变色设计
- 流畅的动画过渡
- 响应式布局（支持手机、平板、桌面）
- 完整的表单验证
- 操作反馈提示（成功/失败消息）

### 5. 企业级组件
- 表格组件: 支持排序、筛选、分页
- 模态框: 用于编辑和确认操作
- 表单组件: 完整的字段验证
- 卡片组件: 整洁的内容组织
- 统计组件: 数据可视化展示

## 📊 数据关系

```
学生 (Student)
  ├─ 选课 → 课程 (StudentCourse → Course)
  ├─ 成绩 (Grade)
  └─ 消息 (Message)

教师 (Teacher)
  ├─ 教授课程 (Course)
  └─ 消息 (Message)

课程 (Course)
  ├─ 教师 (Teacher)
  ├─ 学生选课 (StudentCourse)
  └─ 成绩 (Grade)
```
