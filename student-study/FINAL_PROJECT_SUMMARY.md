## 🚀 快速开始

### 前端启动
```bash
cd student-study
npm install
npm run dev
# 访问: http://localhost:5173
```

### 后端启动 - Docker (推荐)
```bash
cd backend
docker-compose up -d
# 访问: http://localhost:3000
```
