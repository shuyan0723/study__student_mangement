## 🎯 核心技术实现

### 前端框架
- **React 19.1.1** - 最新的React框架
- **TypeScript 5.8** - 完整的类型支持
- **Ant Design 5.11.0** - 企业级UI组件库
- **React Router 6.20.0** - 完整的路由系统
- **Zustand 4.4.1** - 轻量级状态管理
- **Vite 7.1.0** - 极速构建工具

### 状态管理架构
```
AuthStore (认证状态)
  ├─ user (当前用户)
  ├─ token (JWT Token)
  ├─ isAuthenticated (认证状态)
  └─ 方法: login, logout, register

DataStore (业务数据)
  ├─ students (学生列表)
  ├─ teachers (教师列表)
  ├─ courses (课程列表)
  ├─ grades (成绩列表)
  ├─ messages (消息列表)
  ├─ studentCourses (选课列表)
  └─ 各类型的增删改查方法
```

### 类型系统
- **User**: 用户基类
- **Student**: 学生用户
- **Teacher**: 教师用户
- **Course**: 课程信息
- **Grade**: 成绩记录
- **Message**: 消息
- **StudentCourse**: 学生选课关系

### 路由系统 (14+ 个路由)
```
/login                     - 登录页
/dashboard                 - 首页仪表板

/student/grades           - 学生成绩查询
/student/courses          - 学生选课管理
/student/messages         - 学生在线交流

/teacher/grades           - 教师成绩管理
/teacher/classes          - 教师班级管理
/teacher/messages         - 教师在线交流

/admin/students           - 管理员学生管理
/admin/teachers           - 管理员教师管理
/admin/courses            - 管理员课程管理
/admin/grades             - 管理员成绩管理
/admin/settings           - 管理员系统设置

/profile                  - 个人资料页
```
